dotfiles
========

#### Kodamera

`km-vci` cd into folder

`km-dev` mount and cd dev server

`km-stage` ssh stage server


#### Projects

`dotfiles` cd into folder


#### Unix

`..` up one level

`...` up two levels

`e` edit (uses Sublime Text)

`c` clear window

`r` refresh `bash_profile`


#### Git

`gs` git status

`ga` git add

`gc` git commit

`gl` git pull

`gp` git push
